📊 SQL Data Analysis – Practice Repository
🚀 Overview

This repository contains hands-on SQL practice based on a realistic sales dataset.
It focuses on solving business-oriented problems and improving analytical thinking using SQL.

The exercises cover core to advanced SQL concepts commonly used in data analysis and real-world business environments.


📁 Dataset

File: sales_data_200_rows.csv

Records: 200 rows

Columns: 10

Designed to simulate real business scenarios for analytical practice


🛠 Technologies Used

MySQL 

SQL (Core + Advanced)

Spreadsheet tools for data preview

GitHub for version control and project hosting


🔎 Practice Areas

This repository includes practice on:

Data aggregation and grouping

Subqueries and correlated queries

Joins and relational analysis

Conditional logic using CASE statements

CTE (Common Table Expressions)

Window functions (RANK, ROW_NUMBER, LAG, etc.)

Running totals and growth calculations

Data cleaning and duplicate detection
